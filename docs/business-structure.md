```
│  CROSSOVER DEVELOPMENT GROUP LLC    │
│  (Holding Company)                  │
├──────────────────┬──────────────────┤
│     ▼ owns       │      ▼ owns     │
├──────────────────┼──────────────────┤
│ FREE STATE       │ DREAMRIVER       │
│ DEFENSE LLC      │ TECHNOLOGIES LLC │
│ (Operating Co.)  │ (Operating Co.)  │
│                  │                  │
│ Split: 33/33/33  │ Split: 50/50    │
│                  │ Owns its own IP  │
│ Firearms         │ Dream journal    │
│ training         │ SaaS app         │
│ business         │ dreamriver.io    │
└──────────────────┴──────────────────┘
```

**Key advantage:** Each LLC is compartmentalized — liability in one business cannot reach the others. The trust owns everything, so Justin and Tanya don't personally "own" any assets, including the house. Appears asset-free in a lawsuit scenario.

**Partnership note (DreamRiver Tech):** 50/50 partnership. Operating agreement will be structured so each partner can identify individually for tax purposes or create their own LLC at any point to receive profits.
